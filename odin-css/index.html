An element can have many classes, but only one id
It is recommended not to use an id on one page more than one time
Sometimes you can add certain properties to multiple classes at a time.
This can help avoid repetition of properties.
By using chaining class selectors you can apply certain styles only to elements which have certain multiple classes applied to them.

Group selector formula: [selector1][,][selector2]

You can chain-mix different types of selectors, except tag selectors.

Chain-selector formula: [selector1][no space][selector2]

Descendant combinator formula: [selector1][space][selector2]

Descendant combinators match elements with the last specified tag and with ancestor matching the previous tag.

There is no limit to the amount of ancestors in the descendant combinator, thought it is recommended to avoid long lineages in order to prevent confusion.

Basic Style Properties

"color" property sets an element's text color

"background-color" sets the color of an elements background

Colors can be assigned using several kinds of values.

  • keyword teal
  • transparent
  • HEX
  • RGB
  • HSL

Fonts can be changed using "font-family" property

Fonts fall into one of two categories:

  • "Font family names" -- are enclosed in double quotes and usually starts with a capital letter, since it's a name.
  • generic family names -- never use quotes

Multiple font values can be specified within "font-family" property like so: font-family: ""Times New Roman", sans-serif;

The first value is the primary value the browser will look for first. If it can't be displayed, for some reason, the browser will fallback to the next value in the list and so on.

Size of the font can be changed using font-size property.

To change font weight use font-weight property.

To align a body of text horizontally use text-align tag.

Image height and width

To change height or width of an image, use height and width properties, respectively.

Use keyword auto with either width or height to adjust the chosen paramether automatically in proportion to the other one.

Declare both height and width, as this will reserve the space for the image and prevent other elements from taking the space for themselves before the image has loaded.

Adding CSS to HTML

Method 1: External CSS Styles file

External CSS file is the most preferred method of adding CSS to HTML

File containing CSS is conventionally named styles.css

To link an external styles file place link tag with attributes rel="stylesheet" and href="path-to-file"

Method 2: Internal CSS Styles tag

Alternatively it is possible to declare all CSS properties within a style tag nested in head tag

Method 3: Inline CSS style property

The last approach to adding CSS is placing it in style property of the target tag

This method allows to add CSS styling directly to a tag.

It isn't recommended to use Inline CSS because it can create clutter

Note that Inline CSS overrides styling from any other method, which can be useful in some situations.
